% --- Abstract ---
\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

\GreenLab{} is an intelligent energy management system designed for computer science faculty laboratories. The system combines embedded sensing, automated appliance control, computer-vision occupancy estimation, backend decision logic, role-based monitoring, and manual override through a web dashboard. The hardware layer is centred on an ESP32 microcontroller connected to environmental sensors and a relay module. The artificial-intelligence layer uses camera frames and a YOLO-based model to estimate the number of occupants. The backend layer receives sensor and occupancy data, applies operational rules, stores audit records, and exposes secure REST endpoints. The frontend layer presents real-time status, laboratory assignments, manual controls, and administrative actions through an Angular single-page application dashboard.

The project addresses the limitations of purely manual control and purely schedule-based automation by providing a multimodal, governed approach that integrates authentication, authorisation, override management, transfer approvals, fault handling, and audit logging within a single four-tier architecture.

\vspace{0.5cm}
\textbf{Keywords:} IoT, energy management, smart building, ESP32, YOLO, occupancy detection, laboratory automation, Angular, ASP.NET Core
